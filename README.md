# senior-thesis-transfer-learning-core-loss-modeling
This repository contains the source code for the senior thesis:

"Transfer Learning Based Magnetic Core Loss Modeling"

The project investigates transfer learning strategies for reducing target-domain data requirements while maintaining prediction accuracy in magnetic core loss modeling.

## Repository Structure

├── Experiment1/  # Performance Comparison in the Target Domain
├── Experiment2/  # Effect of Target-Domain Data Size
├── Experiment3/  # Fine-Tuning Strategy Comparison
├── Experiment4/  # Parameter Efficiency of LoRA
└── README.md

## How to Run

1. Prepare the datasets from MagNet Library: https://www.princeton.edu/~minjie/magnet.html.
2. Run scripts for each experiment.
